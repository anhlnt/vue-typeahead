# Vuejsで補完機能

## セットアップ

### Brunch 
```bash
npm install -g brunch
```
### Typeahead 1
```bash
npm install --save vue-typeahead
```
### Typeahead 2
```bash
npm install uiv --save
```
### Typeahead 4
```bash
npm install vue-multiselect
```
## 実行
```bash
npm start
```
